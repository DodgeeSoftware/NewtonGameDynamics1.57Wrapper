Detailed Description

The PhysicsCollisionFrame is a container for a NewtonCollision The PhysicsCollisionFrame class is a wrapper around a NewtonCollision Object. The PhysicsCollisionFrame can used as a collision object on its own but doing so will require you to manually handle collisions.
