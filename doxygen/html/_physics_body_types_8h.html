Detailed Description

PhysicsBodyTypes is a small enumeration which we allocate to a PhysicsBody on creation so that we can know what type it is. This is important because there are some special functions that will only work on certain physics bodies and this wrapper provides one common class interface for physicsbodies.

PBT_COMPOUND 

A convex hull is the same as if you took a 3d object and folded plastic wrap around it, stretching it over every concave surface so that the plastic is flat and wrapping it tighly over covex areas. It forms a shell that has a larger volume but small surface area of polygons to collide and makes physics more efficient to compute

PBT_TREE 

Compound Convex Hull Type Body (means multiple convex hulls bound together into one object)

PBT_VEHICLE 

Tree Type Body (not an actual tree, a collection of polys) Vehicle Type Body
